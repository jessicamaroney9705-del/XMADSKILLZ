# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email security@xmadskillz.dev instead of using the issue tracker.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Security Best Practices

- Always request permissions explicitly
- Validate user input
- Use secure communication channels
- Keep dependencies updated
